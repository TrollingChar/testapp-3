
// 1
assert  a.overlaps(b) == b.overlaps(a)
// операция определения перекрывания коммутативна

// 2
assert  a.flyInto(b, v) == -b.flyInto(a, -v)
// относительность столкновения коллайдеров

// 3
assert  a.overlaps(b) || !(a + a.flyInto(b, v).offset).overlaps(b)
// если перед столкновением коллайдеры не перекрываются, это условие выполняется и после него
